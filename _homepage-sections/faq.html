Frequently asked questions

What's a hackathon?

Hackathons are 'development marathons', where people collaborate intensively over a short period of time to create a working project.

Are there any requirements to attend?

Nope! Anyone can join this hackathon, however software development experience is recommended.

Is hack::peel running online, or in-person?

Due to COVID-19 restrictions and guidelines, this event is running exclusively online.

When is the hackathon?

The hackathon is planned to run from Feb 19 - Feb 21. Check the timetable for more information.

Who can attend?

Any student enrolled in a Peel region high-school can attend this hackathon.

Are there any prizes for winners?

With the help of our sponsors, we will be providing monetary as well as software and API prizes to the chosen winners.

Who owns my project?

You and your team own anything you create. Why not make it open source, on GitHub?

Can I start work before the hackathon begins?

While you can explore ideas, you are not allowed to write code or begin working on the project before the hacking period of the hackathon starts.

Is there a theme for the hackathon?

To allow for as much creative liberty, there is no required project theme.
